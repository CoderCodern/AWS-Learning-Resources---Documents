Question 1 S3
A company needs to store application logs that will be accessed frequently for the first 30 days and then archived. The logs must be retained for 7 years for compliance purposes. Which S3 storage configuration is MOST cost-effective?
Correct Answer: B After the frequent access period (30 days), transitioning to S3 Glacier Flexible Retrieval provides the most cost-effective long-term archival solution while meeting compliance requirements. S3 Standard is best for the first 30 days of frequent access.
Question 2 EBS
A developer needs to create an EBS volume for a production database that requires consistent IOPS performance of 15,000 IOPS. Which EBS volume type should be used?
Correct Answer: B Provisioned IOPS SSD (io2) volumes are designed for mission-critical applications requiring sustained IOPS performance and low latency, making them ideal for production databases with high IOPS requirements.
Question 3 EFS
A developer is designing a solution where multiple EC2 instances across different Availability Zones need to share the same file system concurrently. Which AWS storage service should be used?
Correct Answer: B Amazon EFS is a fully managed NFS file system that can be mounted concurrently by hundreds or thousands of EC2 instances across multiple Availability Zones, making it perfect for shared file system scenarios.
Question 4 Instance Store
A developer needs temporary block-level storage with the highest IOPS for caching data. The data can be lost if the instance stops. Which storage option is MOST appropriate?
Correct Answer: C Instance Store provides the highest IOPS performance and is ideal for temporary storage like buffers, caches, and scratch data. Data is lost when the instance stops or terminates, which is acceptable for this use case.
Question 5 S3
An application uploads files up to 500 GB to S3. Which upload method should be used for optimal performance and reliability?
Correct Answer: B Multipart upload is recommended for objects larger than 100 MB and required for objects larger than 5 GB. It improves performance through parallel uploads and provides better error recovery.
Question 6 EBS
A developer launched an EC2 instance in us-east-1a with an attached EBS volume. The instance needs to be moved to us-east-1b. What is the correct approach?
Correct Answer: B EBS volumes are locked to an Availability Zone. To move to a different AZ, you must create a snapshot of the volume and then create a new volume from that snapshot in the target AZ.
Question 7 S3
A company requires that all objects in an S3 bucket be replicated to another bucket in a different region for disaster recovery. What must be enabled on both buckets?
Correct Answer: B S3 Cross-Region Replication (CRR) requires versioning to be enabled on both the source and destination buckets. This allows S3 to track and replicate object versions.
Question 8 EFS
An application stores user-uploaded files that are accessed frequently for the first week, then rarely accessed afterwards. Which EFS configuration provides the best cost optimization?
Correct Answer: C EFS Lifecycle Management can automatically transition files to the Infrequent Access (IA) storage class after a specified period (e.g., 7 days), providing significant cost savings for files that are rarely accessed.
Question 9 EBS
A database application requires a volume that can deliver up to 64,000 IOPS. Which EBS volume type supports this requirement?
Correct Answer: B io2 Block Express volumes can deliver up to 256,000 IOPS and 4,000 MiB/s throughput, making them suitable for the most demanding database workloads requiring 64,000 IOPS.
Question 10 S3
A developer needs to ensure that objects in an S3 bucket cannot be deleted or overwritten for a period of 5 years for regulatory compliance. Which S3 feature should be used?
Correct Answer: B S3 Object Lock in Compliance mode prevents objects from being deleted or overwritten for a specified retention period. Not even the root user can delete protected objects during the retention period.
Question 11 Instance Store
What happens to data stored on Instance Store volumes when an EC2 instance is stopped?
Correct Answer: C Instance Store volumes are ephemeral storage. Data is lost when the instance is stopped, terminated, or if the underlying hardware fails. Only data on EBS volumes persists.
Question 12 EBS
A company needs to share an EBS volume between two EC2 instances in the same Availability Zone for a clustered application. Which solution should be used?
Correct Answer: A EBS Multi-Attach allows a single Provisioned IOPS SSD (io1 or io2) volume to be attached to multiple EC2 instances within the same Availability Zone, enabling shared access for clustered applications.
Question 13 S3
An application generates millions of small log files daily that need to be stored in S3. The files are accessed randomly but infrequently. Which S3 storage class is MOST cost-effective?
Correct Answer: C S3 Intelligent-Tiering automatically moves objects between access tiers based on access patterns, making it ideal for data with unpredictable access patterns. It has no retrieval fees and monitors access patterns for optimization.
Question 14 EFS
A company needs a file system that can scale to petabytes and be accessed from multiple VPCs in different regions. Which solution meets these requirements?
Correct Answer: A EFS can be accessed from multiple VPCs using VPC peering, AWS Transit Gateway, or VPN connections. EFS scales automatically to petabytes without provisioning.
Question 15 S3
Which S3 feature provides the lowest latency access for frequently accessed data while automatically optimizing costs for data that becomes less frequently accessed?
Correct Answer: B S3 Intelligent-Tiering automatically moves objects between frequent and infrequent access tiers based on changing access patterns, providing millisecond latency and automatic cost optimization without retrieval fees.
Question 16 EBS
Where are EBS snapshots stored?
Correct Answer: B EBS snapshots are stored in Amazon S3 (though you cannot access them directly in your S3 console). They are incremental backups and are regionally scoped.
Question 17 Instance Store
Which statement about Instance Store volumes is correct?
Correct Answer: C and D (both are correct) Instance Store volumes are physically attached to the host computer running the EC2 instance, providing very high performance. Data persists during reboots but is lost on stop/terminate.
Question 18 S3
A developer needs to allow public read access to specific objects in an S3 bucket while keeping other objects private. What is the MOST secure approach?
Correct Answer: B Using object-level ACLs allows granular control over which specific objects are publicly accessible while keeping others private, without exposing the entire bucket.
Question 19 EBS
A company wants to optimize costs for EBS volumes that are used for development and testing. The instances run only during business hours (8 hours per day). What is the MOST cost-effective approach?
Correct Answer: B Creating snapshots and deleting volumes when not in use is most cost-effective. Snapshots are cheaper than keeping volumes provisioned. You only pay for the compressed snapshot storage, not the full volume size.
Question 20 EFS
An application running on EC2 instances requires a POSIX-compliant file system that supports concurrent access from thousands of instances. Which storage service should be used?
Correct Answer: C Amazon EFS is a fully managed, POSIX-compliant file system that can be concurrently accessed by thousands of EC2 instances across multiple Availability Zones using standard file system semantics.
